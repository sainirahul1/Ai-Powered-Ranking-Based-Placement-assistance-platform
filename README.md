# Ai-Powered Ranking-Based Placement Assistance Platform

This platform helps users accelerate their tech careers through AI-driven insights and practice.

## Features

- **AI Roadmap Generator**: Personalized weekly learning paths based on your role, experience, and goals.
- **AI Mock Interview**: Practice with an AI interviewer that provides real-time feedback and evaluation.
- **Placement Insights**: Intelligent guidance to help you land your dream job.

## Tech Stack

- **Frontend**: React, Tailwind CSS, Shadcn/UI
- **Backend**: Node.js, Express
- **Database**: PostgreSQL with Drizzle ORM
- **AI**: OpenAI Integration

## Getting Started

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`

## Deployment

Optimized for deployment on Replit.
